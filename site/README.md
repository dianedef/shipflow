# ShipFlow Site

Astro scaffold for the public ShipFlow website.

## Commands

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```

## Structure

- `src/pages/index.astro` - landing page
- `src/layouts/BaseLayout.astro` - base document shell
- `src/components/` - reusable homepage sections
- `src/styles/global.css` - global visual system
